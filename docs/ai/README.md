### Key Rules

From the [AI Training Guide](./training-guide.md):

1. Always use semantic HTML
2. Always include ARIA attributes
3. Always use Framer Motion with `useInView` for scroll animations
4. Always use TypeScript with explicit types
5. Always optimize images
6. Always include structured data for SEO
7. Always use Tailwind classes
8. Always use color palette CSS variables
9. Always use terminal/code editor aesthetic
10. Always ensure minimum touch target of 44x44px
